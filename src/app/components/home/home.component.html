Order to watch MCU movies

This is a question that many ask lately, what movie goes before what? If a person decides to see them all together now, it is impossible only with the titles and covers to know which one to start.

Although the movies of the heroes separately are more or less independent, there is a small thread that forces you to watch certain movies before others.

It should be noted that Marvel is delivering the movies in phases. Each phase contains several films of the heroes alone that end with one of the Avengers together. Therefore, each Avengers movie can be considered the end of a story arc.
